📊 Análise Estratégica Olist E-commerce 2018
Análise exploratória de dados do marketplace brasileiro Olist, identificando oportunidades de crescimento através de otimização de retenção, logística e satisfação do cliente.

📋 Sobre o Projeto
Projeto desenvolvido para o Tech Challenge 1 - FIAP, realizando análise completa dos dados do marketplace Olist (2016-2018) com foco no ano de 2018.
Objetivo: Identificar oportunidades de crescimento sustentável através da análise de receita, logística, pagamentos e satisfação do cliente.

🚨 Problemas Críticos Identificados
Retenção Inexistente
- 97% dos clientes compram apenas uma vez
- Taxa de recompra: apenas 3%
- 83% da base tem potencial de fidelização não explorado
Desigualdade Logística
- Lead time varia 234% entre regiões (8,8 dias em SP vs. 29,4 dias em RR)
- Estados com maior lead time têm maior ticket médio (oportunidade perdida)
- 74% do tempo está no trânsito (gargalo principal)
Atrasos Destroem Satisfação
- NPS despenca 87 pontos com atraso: +45 (no prazo) → -42 (atrasado)
- Avaliação cai de 4,29★ para 2,57★
